# C12.3: Differential-Privacy Safeguards

> [Back to C12 Index](C12-Privacy.md)
> **Last Researched:** 2026-03-29

## Purpose

Track and enforce privacy budgets to provide formal guarantees against individual data leakage. The Fundamental Law of Information Recovery (Dwork & Roth, 2014) shows that too many adaptive queries destroy privacy — without explicit budget tracking, the cumulative epsilon can grow without limit, degrading privacy guarantees to meaninglessness. As of 2025-2026, the DP tooling ecosystem has matured significantly: Google's VaultGemma (September 2025) demonstrated production-grade DP-SGD for LLMs at 1B parameters with epsilon <= 2.0, proving formal DP guarantees are achievable at LLM scale. DynamoEnhance extends DP-SGD to >7B-parameter models via multi-GPU DeepSpeed integration. Google's user-level DP fine-tuning research (ELS/ULS) shows orders-of-magnitude noise reduction versus prior approaches. PrivCode (NDSS 2026) is the first DP synthesizer for code datasets. However, DP reversal via LLM feedback (2025-2026 research) shows that confidence score analysis can defeat DP protections at the inference layer — API-level mitigations (hiding logits) are needed alongside training-time DP. NIST finalized formal guidelines in early 2025 standardizing DP evaluation.

## Tooling

- **Differential privacy:** Opacus (PyTorch), TensorFlow Privacy (TF), Google DP Library (C++/Java/Go with Python bindings), OpenDP/SmartNoise (Rust core with Python API, Harvard/Microsoft), PipelineDP4j (JVM — Google's newest release for parallelizable DP computations), Diffprivlib (IBM), DynamoEnhance (multi-GPU DP training for >7B-parameter LLMs built on DeepSpeed, supports LoRA + BitsAndBytes + flash attention). Google's VaultGemma (September 2025) demonstrated production-grade DP-SGD for LLMs at 1B parameters with epsilon <= 2.0 — weights available on Hugging Face/Kaggle. Google's user-level DP fine-tuning research (ELS/ULS) shows orders-of-magnitude noise reduction versus prior approaches. PrivCode (NDSS 2026) is the first DP synthesizer specifically for code datasets, using a two-stage DP-SGD + utility-boosting pipeline.
- **Privacy auditing:** DP-Sniper (black-box DP auditing), membership inference attack toolkits (ML-Doctor, privacy meter), canary-based memorization testing
